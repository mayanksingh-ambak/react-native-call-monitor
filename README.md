# react-native-call-monitor

Library helps to monitor the outgoing call from user device

## Installation

```sh
npm install react-native-call-monitor
```

## Usage


```js
import { multiply } from 'react-native-call-monitor';

// ...

const result = await multiply(3, 7);
```
